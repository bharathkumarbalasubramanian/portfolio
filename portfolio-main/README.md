# BHARATHKUMAR B | AI Product Manager Portfolio

Modern, high-performance portfolio dashboard built for the next generation of AI Product Leadership.

## 🚀 Features
- **Full-Stack Architecture**: Powered by **FastAPI (Python)** for scalable backend logic and AI integration.
- **Modern UI**: Implemented with **Tailwind CSS**, highlighting professional AI metrics and a glassmorphic aesthetic.
- **Dynamic Content**: Hero statistics and featured projects are fetched dynamically via API.
- **Interactive Visuals**: 3D background elements integrated with **Three.js**.
- **Dark/Light Mode**: Fully responsive, theme-aware design.

## 🏗️ Project Structure
- `/frontend`: The web interface and creative assets.
- `/backend`: The Python server providing a robust API layer for AI features.

## 🛠️ Tech Stack
- **Frontend**: HTML5, JavaScript (ES6+), Tailwind CSS, Three.js.
- **Backend**: Python 3.9+, FastAPI, Uvicorn.

## 🏃 Local Setup

### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `uvicorn main:app --reload`

### Frontend
- Open `/frontend/index.html` in any modern browser.
