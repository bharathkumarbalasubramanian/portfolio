### Step 1: Install Dependencies
```bash
cd your-portfolio-folder
npm install --save-dev tailwindcss postcss autoprefixer
```

Or just run the provided script:

### Step 2: Generate CSS
**For development (auto-reload on changes):**

**For production (minified):**
```bash
npm run build
```

### Step 3: Done! 🎉
Your `output.css` file is generated and automatically linked in `index.html`

## 🐛 Troubleshooting

### Problem: `output.css` not being created
**Solution:**
```bash
# Reinstall everything
rm -rf node_modules package-lock.json
npm install --save-dev tailwindcss postcss autoprefixer
npm run build
```

### Problem: Still seeing the CDN warning
**Solution:**
1. Make sure `index.html` has `<link rel="stylesheet" href="output.css">`
2. Check that the old `<script src="https://cdn.tailwindcss.com">` is removed
3. Clear browser cache (Ctrl+Shift+Delete)

### Problem: Styles not applying
**Solution:**
1. Make sure `npm run build` or `npm run dev` has been run
2. Check if `output.css` exists in your folder
3. Hard refresh the page (Ctrl+F5)

### Problem: Need to modify Tailwind config
**Solution:**
1. Edit `tailwind.config.js`
2. Re-run `npm run build`
3. Hard refresh browser (Ctrl+F5)

## 🚀 Deployment Guide

### For Hosting (Netlify, Vercel, GitHub Pages, etc):

1. **Before deploying**, run production build:
```bash
npm run build
```

2. **Upload these files:**
   - ✅ `index.html`
   - ✅ `output.css` (important!)
   - ✅ `profile-picture.png`
   - ✅ `script.js`
   - ✅ `three-scene.js`
   - ✅ `style.css` (optional, but keep it)

3. **Do NOT upload:**
   - ❌ `node_modules/` (too large)
   - ❌ `package.json` (if you don't want others to rebuild)
   - ❌ `input.css` (already compiled into output.css)
   - ❌ Config files (tailwind.config.js, postcss.config.js)

4. **Or let the host handle it:**
   Most modern hosts (Vercel, Netlify) automatically run `npm install` and `npm run build` for you!

## 💡 Pro Tips

### Customize Colors
Edit `tailwind.config.js`:
```javascript
colors: {
  primary: '#your-color',
  // ... more colors
}
```
Then run `npm run build`

### Add New Fonts
1. Import from Google Fonts (already done)
2. Update `tailwind.config.js` fontFamily
3. Run `npm run build`

### Extend Tailwind
Add custom utilities in `input.css`:
```css
@layer components {
  .btn-custom {
    @apply px-4 py-2 rounded-lg font-bold;
  }
}
```
Then run `npm run build`
